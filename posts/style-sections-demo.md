This post demonstrates the new style sections feature.

# Regular Content

This content appears with normal styling.

~style~
background-color: #e8f4fd; padding: 20px; border-left: 4px solid #2196F3; margin: 20px 0;
~style~

## Light Blue Section

This content is wrapped in a light blue container with padding and a blue left border.

You can include any content here:
- Lists work fine
- **Bold text** too
- And regular paragraphs

~style~
background: linear-gradient(45deg, #ff6b6b, #feca57); color: white; padding: 25px; border-radius: 15px; text-align: center; margin: 30px 0;
~style~

# Gradient Section

This section has a colorful gradient background with white text and center alignment!

~style~
background-color: #2c3e50; color: #ecf0f1; padding: 15px; font-family: monospace; border: 2px solid #34495e;
~style~

## Dark Code-like Section

This section looks like a terminal or code editor with:
- Dark background
- Light text
- Monospace font
- Subtle border

~style~
background-color: #fff3cd; border: 1px solid #ffeaa7; padding: 15px; border-radius: 5px;
~style~
